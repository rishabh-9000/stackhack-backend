## Prerequisite

Node.js 12+ => https://nodejs.org/en/download/
MongoDB 4+ => https://www.mongodb.com/try/download/community

## Get the project

```bash
$ git clone https://github.com/rishabh-9000/stackhack-backend.git
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Note: MongoDB local server should be running.
